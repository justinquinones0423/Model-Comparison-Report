# Week 4: Model Comparison

Tested 4 AI models on 5 cybersecurity text samples to evaluate their suitability
for the ingestion component of our security alert triage bot capstone project.
## Models Tested
- HF Sentiment (distilbert-sst-2)
- HF Zero-Shot (bart-large-mnli)
- HF NER (bert-large-NER)
- Groq Llama 3 8B
## Finding
Recommended Groq Llama 3 8B for the ingestion component because that model labeled a category and an in depth analysis for each cybersecurity text sample.
See `report.md` for full analysis.
